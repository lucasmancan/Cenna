# Cenna

Free, lightweight macOS menu-bar app for screenshots and screen recording — native capture, a built-in annotation editor, MP4/GIF recording with system audio, mic and webcam, and strict memory discipline.

**[⬇︎ Download the latest release](https://github.com/lucasmancan/Cenna/releases/latest)**

- Requires macOS 13+
- Free to use
- More info: [mancan.digital/products/cenna.html](https://mancan.digital/products/cenna.html)

## Install

1. Download the `.dmg` from [Releases](https://github.com/lucasmancan/Cenna/releases/latest).
2. Open it and drag **Cenna** to Applications.
3. First launch: right-click the app → **Open** (the build is not yet notarized, so Gatekeeper asks once).
